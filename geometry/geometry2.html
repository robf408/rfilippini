To get started, shift-click on the canvas. This should create a point. Try dragging the point. Now alt-click on the point. It should get smaller. It is now in the deselected state. Alt-click on it again to reselect the point. Now shift-click somewhere else to create another point. Then Choose "Line" from the Action menu. You should see a line through your points. The points are still moveable of course. The line is currently selected. You can alt-click on it to deselect it. If you leave it selected and shift-click to create a new point, the new point will be constrained to the line. Similarly, points can be constrained to circles or conics. Depending on what is currently selected, certain actions are permitted. Here is a list explaining the actions in the Action menu; each entry lists the types of the selected elements followed by "=>" followed by a description of the result.

  • Point
    • Point, Point => MidPoint
    • Line, Line => Intersection
  • Line
    • Point, Point => Line through 2 points
    • Line, Point => Line perpendicular to line through point
    • Conic, Point => Line tangent to conic through point - conic must contain selected point
    • Line, Line, Point => Angle Bisector - both lines contain selected point, rotate first line CCW halfway to second line.
  • Circle
    • Point, Point => Circle (center must be selected first)
  • Conic
    • Point, Point, Point, Point, Point => Conic through 5 points

Colors can be applied to selected elements. The hide button causes selected elements to be hidden. The show button shows all hidden elements and causes them to be selected. The < button steps backwards and the > button steps forward through the construction. The Configurations menu lists various prebuilt configurations. Many of these demonstrate interesting theorems in geometry. The Custom entry in the Configurations menu shows a text area and 2 buttons labeled "set text" and "set diagram". If you click "set text", the text area is filled with a text representation of the current diagram. The "set diagram" button updates the diagram based on the text. Each element can either be in "clean" mode or "trace" mode. The default of clean mode means that when the element is redrawn in some new location, it is first erased. Clicking on the trace button causes the selected elements to enter trace mode. Clicking on the clean button causes all elements to be put into clean mode. For an interesting application, choose the Dual of Euler Line configuration, select the blue line, press trace, then move the control point around the circle. There is also a maximize/minimize button in the upper right-hand corner. If you want to recenter your drawing, press the alt and shift keys while dragging the canvas.
